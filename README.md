# Requirements
-make sure your port 3000 is free
-open a terminal

```sh
npm i
npm install -g parcel-bundler
npm install -g browser-sync
```

# Running
- In the same terminal

```sh
parcel watch app.js
```

- In another terminal
```sh
browser-sync start --server --files '.'
```

# You should see the application running now, Congrats ! :)
